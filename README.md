# Vue chat

Créer l'interface d'un chat qui utilise Vue.

### Démarrage

Pour lancer le projet, il est nécessaire d'utiliser un serveur http.

### Architecture

- `index.html`
- `App.js`
- `components/`
	- `Chat.js`
	- `Message.js`
	- `Notification.js`
	- `User.js`
	- `Users.js`
- `js/`
	- `burger.js`
	- `faker.min.js`
- `style/`
	- `favicon.svg`
	- `nomessage.svg`
	- `style.css`
